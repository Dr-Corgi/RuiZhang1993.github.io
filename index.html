[SMP-SA2017] 终身机器学习LML

2017-06-11

什么是终身机器学习(LML)？

传统的机器学习通常是孤立的单任务机器学习算法：针对某个特定任务，给定一份数据集，运行某些机器学习算法，训练得到一个模型。
这种传统的机器学习方法通常不会考虑到过去的知识。也就是说，每次遇到新的问题，传统机器学习方法都是从零开始训练一个新模型。这种方式被称为“机器学习1.0”。
“机器学习1.0”存在着一些明显的缺点，例如每次构建模型都是一次孤立学习过程，过往学习到的知识既没有被存储，也不能被重新获取。同时，也正因为知识不能够被存储，每次训练新模型时我们总是需要大量的新的训练数据。

2017-05-27

标题：A neural probabilistic language model
作者：Bengio Y, Ducharme R, Vincent P, et al.
来源：Journal of machine learning research, 2003, 3(Feb): 1137-1155.
论文地址：http://www.jmlr.org/papers/volume3/bengio03a/bengio03a.pdf

这篇文章是最早使用神经网络来构建概率语言模型的工作，也是现在比较流行的“词向量”最早的相关工作之一。文章提出了利用一个三层的神经网络来学习概率语言模型，通过给定的上文窗口，模型预测下一个位置的输出词语，在经过足够次数的训练迭代后，神经网络将学习到语言的规律，而词向量将作为模型的副产品得到。

什么是卷积神经网络

简单来说，卷积神经网络是指那些至少在网络的一层中使用卷积运算来替代一般的矩阵乘法运算的神经网络。
常用的卷积网络通常由很多层的卷积运算和池化运算堆叠起来组成。这篇文章就简单来讲讲什么是卷积运算、池化运算，以及为什么要使用卷积运算。

2017-05-19

标题：A Neural Conversation Model
作者：Oriol Vinyals, Quoc Le
来源：ICML Deep Learning Workshop 2015
论文地址：https://arxiv.org/abs/1506.05869

这篇文章是最早将神经机器翻译中的seq2seq生成技术应用到人机对话任务的工作之一。文章提出了一个端到端(end-to-end)的对话生成模型，这里的端到端可以理解为：模型的输入是一个问答对中的问句，而输出则是问答对中相应的回答。模型摒弃了对答句的复杂组织过程，而是利用语言模型不断预测下一个输出词语，直到输出句子终止符的方式来生成回答。

Mac下卸载/更新numpy

2017-01-19

今天安装PyTorch的时候发现numpy模块需要更新，使用pip install -U numpy的时候发现有些操作被系统禁止了（不是sudo权限的原因）。百度了一下发现是系统的安全机制引起的。具体的解决方案如下：

  1. 重启电脑，重启时按住command+R
  2. 进入“OS X工具程式”界面，打开终端，输入csrutil，重启电脑即可。
